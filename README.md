# Heart Disease Prediction Using Machine Learning

## Project Overview
This project predicts the possibility of heart disease using patient medical data.

## Technologies Used
- Python
- Machine Learning
- Flask
- HTML
- CSS

## Machine Learning Models
- Logistic Regression
- KNN
- Random Forest

## Features
- Patient data input
- Disease risk prediction
- Web-based user interface

## Dataset
Heart Disease Dataset

## How to Run

Install requirements:

pip install -r requirements.txt

Run application:

python app.py

Open browser:

http://127.0.0.1:5000
